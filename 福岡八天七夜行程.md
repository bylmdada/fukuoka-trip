## 🚌 福岡近郊行程推薦（親子 / 雨天備案）

> [!TIP]
> 以下行程可根據天氣、體力狀況彈性調整，適合親子同遊或作為雨天備案

### 1️⃣ 太宰府＋太宰府遊樂園（強推親子 👨‍👩‍👧）

**亮點**：太宰府天滿宮步行區平坦、有小吃（梅枝餅）、商店街，大人小孩都好走。

| 項目 | 資訊 |
|------|------|
| 位置 | 太宰府天滿宮旁 |
| 設施 | 約 20 種，小火車、旋轉木馬、迷你雲霄飛車、腳踏天際車 |
| 適合年齡 | 2 歲以上即可搭乘多數設施 |
| 建議時長 | 半日（可搭配天神購物半日）|
| 雨天適合 | ⭕ 可逛商店街、吃東西 |

**行程示意**：
```
博多站 →（西鐵電車約 40 分）→ 太宰府天滿宮＆遊樂園 → 傍晚回天神吃晚餐逛藥妝
```

### 2️⃣ 海之中道海濱公園＋水族館（放電一整天 🐬）

**亮點**：腹地超大，有動物森林、小型動物園、花園、主題遊具、草地，親子放電首選。

| 項目 | 資訊 |
|------|------|
| 兒童廣場設施 | 大型彈跳雲（鯨魚雲）、長溜滑梯、水上遊具、低齡兒童遊具 |
| 搭配景點 | Marine World 海之中道水族館（海豚、海獅表演）|
| 交通方式 | 博多港或香椎出發，可租腳踏車或搭園內小火車 |
| 建議時長 | 一整天 |
| 親子評價 | ⭐⭐⭐⭐⭐ 非常高 |

**行程示意**：
```
博多港或香椎出發 → 海之中道海濱公園 → 海之中道水族館 → 傍晚回博多
```

### 4️⃣ 柳川運河遊船（節奏慢、適合放鬆 🛶）

| 項目 | 資訊 |

**行程示意**：

西鐵福岡（天神）→ 柳川遊船 → 午餐鰻魚飯 → 回福岡市區

| 最佳時機 | 精神較好的日子 |

**行程示意**：
```
博多站 →（新幹線/特急）→ 小倉 → 門司港懷舊區 → 九州鐵道紀念館 → 返回福岡
```
